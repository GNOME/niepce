Niepce Digital
==============


# What is this?

Niepce Digital is a digital photography software. Its goal is to
provide asset management and image processing and editing.

It is currently work in progress.

# To build

See [build](doc/build.md).

Please see the [contributing guide](doc/contributing.md).

# Getting the code

Obviously you have it.

The most recent code is hosted on GNOME's gitlab server. See:

https://gitlab.gnome.org/GNOME/niepce

# Getting in touch
